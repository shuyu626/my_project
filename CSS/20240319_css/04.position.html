position: relative 相對定位

當元素為相對定位時

  • 使用位移屬性(top、right、bottom、left)，會以自己原來的位置做位移，保留自身原來的空間，不會被其他原素佔據
  • 沒有任何位移屬性，只有設定 position:relative，會留在原地不動，relative 元素常做為 absolute 子元素位移目標依據
  • 進入定位層，可以設定 z-index

position: absolute

當元素為絕對定位時

  • 沒有任何位移屬性，該元素脫離正常流向，進入定位層，後方元素會流動至絕對定位元素的下方(被遮住)
  • 若祖先元素找不到定位點(absolute、relative、fixed)，最後會以 html 根元素做為位移的目標
