各尺寸的欄位分布狀況

xs => 1 欄，可以不寫 .col-12，表示一般 div 垂直往下排列

md => 2 欄

lg => 3 欄

xl => 4 欄


xs => 1、3 欄平均分配空間，但是 2 欄是一般 div 佔滿容器寬度，會將 1、3 切開，呈現由上至下垂直排列

md => 1、3 欄平均分配空間，2 欄會是內容文字的寬度，在 lg 尺寸之前 1、3 欄都會維持 .col 平均分配空間，所以剩餘的空間會均分給1、3欄

lg => 1、3 各分 2 等份，2 欄依舊維持內容寬度，會有剩餘空間，由於 .row 他本身是彈性盒，可以將三欄(彈性項目)對齊的主軸 center(justify-content-md-center)
